# n8n Workflows for ARIA

This directory stores automation workflows used by ARIA, such as:
- Scheduled ingestion of earnings call transcripts and SEC filings.
- Data enrichment jobs (symbol tagging, sector classification).
- Triggering vector upserts into Pinecone.
- Alerting and report delivery workflows.

Import `sample_market_ingestion_workflow.json` into n8n and connect credentials.
